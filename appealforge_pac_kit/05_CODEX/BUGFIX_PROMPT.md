# BUGFIX PROMPT

Fix the current AppealForge PAC issue.

Process:
1. Reproduce the error.
2. Identify root cause.
3. Make the smallest safe fix.
4. Add or update a test.
5. Run relevant checks.
6. Update `.codex/progress.md`.
7. Continue to the next known issue if time remains.

Do not paper over errors by removing core requirements:
- citation validation;
- audit logging;
- reviewer statuses;
- packet versioning;
- synthetic demo guardrails.
