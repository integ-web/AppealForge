# SPRINT 02 — UI Design Polish

Goal: make the app feel premium and lovable.

Build:
- Branded login page.
- Dashboard with KPI cards, urgent queue, case cards, payer pattern panel.
- Case overview page.
- Shared components: Badge, DeadlineBadge, EvidenceStatusRail, SourceCitationPill, PacketPaper, EmptyState, LoadingSkeleton.
- Responsive layout.
- Accessibility focus states.

Acceptance:
- App no longer looks bland.
- Login page could be used in a sales demo.
- Dashboard hierarchy is immediately clear.
- Empty/loading/error states exist.
