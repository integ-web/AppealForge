# SPRINT 04 — Evidence Map + Packet Builder

Goal: make the evidence map and packet editor the product's core demo.

Build:
- Evidence map cards.
- Source quote drawer.
- Missing evidence checklist.
- Reviewer status workflow.
- Packet markdown generator.

- Packet editor.
- Packet version history.
- Draft/approved state.

Acceptance:

- Packet uses only cited/reviewed evidence.
- Unsupported clinical sentences are blocked.
- Packet preview looks like warm paper.
