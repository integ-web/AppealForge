# SPRINT 03 — Document + AI Workflow Demo

Goal: create deterministic synthetic document flow that simulates extraction and AI evidence mapping.

Build:
- Document upload UI.
- Local document metadata.
- Seeded extracted pages.
- Document classification function.
- Denial parser review page.
- Denial parser schema.
- Criteria generation for 3 case types.
- Evidence extraction mock using seeded text.
- Source citations.

Acceptance:
- User can move from case to denial parser to evidence map.
- Every parsed denial field has a citation where possible.
- Evidence map can be regenerated deterministically.
