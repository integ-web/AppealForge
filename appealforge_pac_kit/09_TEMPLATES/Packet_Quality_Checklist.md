# Packet Quality Checklist

Before export:

- [ ] Denial reason matches notice.
- [ ] Appeal deadline confirmed.
- [ ] Requested service is correct.
- [ ] Every clinical claim has citation.
- [ ] Evidence table includes document/page/quote.
- [ ] Missing evidence is resolved or noted.
- [ ] Physician statement is complete if needed.
- [ ] Attachment index matches files.
- [ ] Draft watermark is present unless approved.
- [ ] No real PHI in demo mode.
- [ ] Packet approved by authorized reviewer.
